##### 个人博客管理系统

###### 前端
###### react18 + vite + antd
###### 后端
###### java + SpringBoot + MyBatis + JWT + Redis + mysql


###### Build Setup

```bash
# clone the project
git clone https://github.com/Hedongliang0518/blog_admin.git

# enter the project directory
cd blog_admin

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:9999
